# Anirbaan Sarkar — Portfolio v2

A high-performance, design-first personal portfolio focused on product engineering and AI-powered interfaces. Built with a minimalist, typography-driven aesthetic and premium interactive details.

**Live Demo:** [anirbaansarkar.vercel.app](https://anirbaansarkar.vercel.app/)

---

## 🎨 Design Philosophy

This portfolio is built on a "Product-First" philosophy, transitioning from a simple student showcase to a systemic, editorial experience. It emphasizes high contrast, intentional spacing, and a strict 8px/10px grid system to achieve a professional, premium feel.

- **Typography Centric**: Utilizing oversized headlines and justified text for an editorial, publication-style layout.
- **Minimalist Palette**: Solid black on an off-white foundation (`#F2F1EC`) to ensure focus remains on the content.
- **Micro-Interactions**: Subtle, organic movement integrated into the system rather than layered on top.

## 🚀 Key Features

- **Sequential Preloader**: A cinematic percentage-based entry animation that sets the tone for the experience.
- **Advanced Interaction System**:
  - **Magnetic Pull**: Interactive elements (links, buttons, project cards) subtly attract the cursor.
  - **Typography Reaction**: The custom cursor lens expands and reacts when hovering over primary headlines.
  - **Grid Proximity**: A subtle 'light' following the cursor that highlights the underlying grid architecture.
- **Featured Project Architecture**: Dedicated layouts for high-impact work like **SubsTrack**, emphasizing problem-solving and technical stack.
- **Optimized Performance**: 
  - Efficient rendering using React and hardware-accelerated transforms via Framer Motion.
  - Fully responsive and accessibility-aware.
  - Custom cursor and heavy interactions automatically disabled on mobile for native performance.
- **SEO Ready**: Meta tags, sitemaps, and structured heading hierarchy for maximum visibility.

## 🛠️ Technology Stack

- **Core**: React 18
- **Styling**: Tailwind CSS (Minimal & Utilitarian)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel

## 📂 Project Structure

- `src/components`: Reusable UI elements (CustomCursor, Magnetic, Navbar, etc.)
- `src/sections`: Logical blocks of the primary view (Hero, About, Projects, etc.)
- `src/pages`: Detailed project case studies.
- `src/utils`: Data models and content maps.

## 🛠️ Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anonically22/portfolio-v2.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
